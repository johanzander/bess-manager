# Changelog

All notable changes to BESS Battery Manager will be documented in this file.

## [2.5.0] - 2025-11-06

### Added

- Home Assistant add-on repository support
- Proper repository structure for direct GitHub installation
- Build configuration for multi-architecture support
- Documentation for add-on installation and configuration

### Fixed

- Repository validation for Home Assistant add-on store integration

### Changed

- Restructured repository to support direct installation from GitHub
- Updated build process to work with Home Assistant add-on system

## [2.4.0] - Previous Release

### Features

- Warning banner for missing historical data
- Optimization can start from below minimum SOC with warning
- Fixed issues with savings and grid import columns in savings view
- Improved documentation and logging
